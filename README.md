# Dream Image Animation using Python (GIF)

This project generates an image-based animation (GIF) from user imagination using Python 3.12.
Instead of creating a video, the program creates multiple image frames and combines them into a smooth animated GIF, representing a dream or imagination visually.

# Features

Takes text input from the user (dream or imagination)
Generates multiple image frames
Converts frames into an animated GIF
Runs smoothly on Python 3.12
Lightweight and beginner-friendly


# Technologies Used

Python 3.12
Pillow (PIL) – for image creation and GIF animation


# Installation

Install the required library using pip:
pip install pillow


# How to Run

1. Clone the repository:
git clone https://github.com/your-username/your-repo-name.git

2. Navigate to the project folder:
cd your-repo-name

3. Run the program:
python dream_animation_gif.py

4. Enter your imagination when prompted.


# Output

frames/ → Contains generated image frames
dream_animation.gif → Final animated image (GIF)
You can open the GIF in any browser or image viewer.

# Example Input

I was flying above a glowing city with stars in the sky.
